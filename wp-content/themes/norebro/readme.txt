Norebro is flexible and high performance multipurpose WordPress Theme for personal and corporate use. It comes with the most popular WordPress page builder WPBakery Page Builder (formerly Visual Composer). Create gorgeous pages and layouts fast and easy without coding.

Author: Colabrio
Author URI: https://clbthemes.com
Docs Website: https://docs.clbthemes.com/norebro
Help Center: https://colabrio.ticksy.com

# Installation

## Manual installation:
### Upload the "norebro" folder to the "/wp-content/themes/" directory;

# Activation and Usage

## 1. Activate the theme through the "Appearance > Themes" menu in WordPress;
## 2. Check Theme Settings menu to change the appearance using pre-made options;

# Custom Logo

This theme supports custom logo for the header part;
You can change or remove it from "Theme Settings > Branding > Website Logo";
